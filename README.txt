■使用法

	・修論本文である main.tex と、表紙である title.tex の内容を
	  自分のものに書き換える

	・和文要旨である abst.tex の内容を自分のものに書き換える

■修論執筆に関しての注意事項

	・章構成を十分に練った上で書き始める
		- 提出直前に章構成を変えるのは至難の技

	・読者の立場に立った解りやすい表現を心がける
		- 主述が合わない文を書かない
		- 複雑な部分は図や表を活用する
		- 読者は２年前の自分を思い浮かべると良いかも

	・各章ではしっかりと結論（言いたいこと）を述べる

	・身体を壊さない程度に提出締切直前までひたすら頑張る
